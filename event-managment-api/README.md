📖 Description
Event Management API est une API REST développée avec Node.js, Express et Console Neon permettant de gérer des événements.
Elle permet de :
➕ Créer des événements
📄 Lister les événements
🔍 Voir detail un événement
❌ Supprimer un événement
👥 Gérer les participants
📦 Installation
1. Cloner le projet
git clone url
2. Installer les dépendances
cd event-managment-api
npm intall ou yarn 
3. Lancer app
npm run dev
